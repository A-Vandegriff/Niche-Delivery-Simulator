This Java project simulates a day in the life of me as a truck fleet worker working for a niche company that delivers liquid to designated fields. Along with simulate a Day in the life, this program will beable to calculate the ideal route, the estimated time a field will take to finish, and how many days a job will take in total. I'll be using simualted amounts of gallons the farm wants per field, the time the routes take, how many trucks are in the fleet, the rig set up, break downs, distance from shop, etc.

simulation stages:
  -night before
  -morning of work
  -workday
  -end of work


I run this program using a Visual Studio Code and WSL
    -import files into your IDE
    -Run javac Main.java
    -Run java Main.java
